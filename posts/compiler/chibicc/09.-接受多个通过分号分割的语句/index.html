接受多个通过分号分割的语句#

C 语言是通过一系列的语句组合的

每个语句都进行代码生成

之间同样通过接口关联

program    ::= stmt*
stmt       ::= expr-stmt
expr-stmt  ::= expr ";"
expr       ::= equality
equality   ::= relational ("==" relational | "!=" relational)*
relational ::= add ("<" add | "<=" add | ">" add | ">=" add)*
add        ::= mul ("+" mul | "-" mul)*
mul        ::= unary ("\*" unary | "/" unary)*
unary      ::= ("+" | "-") unary
             | primary
primary    ::= "(" expr ")" | NUM

代码生成还是基于 AST 的

  • 这里每一个 stmt 就是一个 AST
  • 所以用了一个 Node 数组来存储所有的 stmt
