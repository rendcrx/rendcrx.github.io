变量：内存的一个别名

  • 环境中的一个映射
  • 变量本质就是一个映射
program    ::= stmt*
stmt       ::= expr-stmt
expr-stmt  ::= expr ";"
expr       ::= assign
assign     ::= equality ("=" assign)?
equality   ::= relational ("==" relational | "!=" relational)*
relational ::= add ("<" add | "<=" add | ">" add | ">=" add)*
add        ::= mul ("+" mul | "-" mul)*
mul        ::= unary ("\*" unary | "/" unary)*
unary      ::= ("+" | "-") unary
             | primary
primary    ::= "(" expr ")" | IDENT | NUM
